# Review Pack - LMS Teacher/Student

Bộ tài liệu review nhanh cho bài toán:
- Vai trò: `Teacher`, `Student`
- Teacher tạo lớp, môn học, bài học, câu hỏi
- Student xem bài, làm bài, nộp bài

## Danh sách file

1. [`01-overview.md`](./01-overview.md) - Mục tiêu, phạm vi, module chính
2. [`02-use-cases.md`](./02-use-cases.md) - Use case theo vai trò
3. [`03-data-model.md`](./03-data-model.md) - Mô hình dữ liệu + ERD
4. [`04-user-flow.md`](./04-user-flow.md) - Luồng thao tác teacher/student
5. [`05-roadmap-v1.md`](./05-roadmap-v1.md) - Lộ trình triển khai theo phase

## Cách đọc nhanh

- Bắt đầu từ `01-overview.md`
- Xác nhận nghiệp vụ ở `02-use-cases.md`
- Chốt DB/API từ `03-data-model.md`
- Chốt UI flow ở `04-user-flow.md`
- Lên kế hoạch sprint theo `05-roadmap-v1.md`

## Lưu ý

- Các sơ đồ dùng `Mermaid` để dễ sửa.
- Nếu cần, có thể render sang ảnh bằng plugin Markdown của IDE.
